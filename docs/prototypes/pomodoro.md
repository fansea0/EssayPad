# 番茄钟 - 原型图 v1

> 极简 · 小窗口 · 大圆环 + 大时间

---

## 1. 任务列表(入口)

每条任务右侧 hover 出现「▶ 开始专注」按钮。任务行也加统计:`🍅 N · ⏱ X`。

```
┌─────────────────────────────────────────────────────────────┐
│ 任务 · 今日 2/3 完成        [＋ 新建任务]                   │
│ [今天] [昨天] [本周] [全部]                                │
├─────────────────────────────────────────────────────────────┤
│ 🔥 紧急 (1)                                                 │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │▌│ ● 重构登录页  🔥 紧急                                │ │
│ │▌│ ████████░░ 75%                                       │ │
│ │▌│ 今日 2 / 4 番茄 · ⏱ 1h 15m    [   ▶ 开始专注   ]    │ │  ← 新加
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ⭐ 重要 (1)                                                 │
│ ┌─────────────────────────────────────────────────────────┐ │
│ │▌│ ● 修 toggle 闪烁     ⭐ 重要                         │ │
│ │▌│ ██░░░░░░░░ 25%     [   ▶ 开始专注   ]                │ │
│ └─────────────────────────────────────────────────────────┘ │
│                                                              │
│ ───────────────                                             │
│ ✅ 已完成 (1)                                                │
└─────────────────────────────────────────────────────────────┘
```

**交互:**
- hover 任务行 → 右侧出现「▶ 开始专注」按钮(蓝色文字按钮)
- 点击 → 弹启动弹窗(#2)
- 「🍅 N」:`N` 是今天完成的番茄数;`⏱ X` 是累计专注分钟

## 2. 启动弹窗(选预设)

小弹窗,大字号时间显示当前计划,4 个预设可选。

```
┌──────────────────────────────────────────┐
│  🍅 专注  ·  重构登录页              [×] │
├──────────────────────────────────────────┤
│                                          │
│            25 : 00                        │  ← 64pt 粗体 Mono
│          默认 25 分钟                    │  ← 12pt 灰
│                                          │
│    ┌──────┬──────┬──────┬──────┐         │
│    │  15  │  25  │  45  │ 自定 │         │  ← 选中蓝底
│    │  分  │  分  │  分  │  ·  │         │
│    └──────┴──────┴──────┴──────┘         │
│                                          │
│   ── 任务 ──                              │
│   🔥 紧急 · 进度 75%                    │
│   今日已 2 个番茄 · 累计 1h 15m         │
│                                          │
│   ┌──────────────────────────────────┐   │
│   │           开 始 专 注             │   │  ← 大主按钮
│   └──────────────────────────────────┘   │
│                                          │
└──────────────────────────────────────────┘
```

**交互:**
- 点预设 15/25/45 → 时间立即更新
- 「自定」→ 滑块或数字选择(1-99 分钟)
- 选中预设:蓝底白字;未选中:灰底黑字
- 「开始专注」主按钮按下后,**这个弹窗直接变成倒计时窗口**(转场平滑)

**窗口规格:** 360 × 400,圆角 16

## 3. 倒计时窗口(主战场 — 极简小窗)

**极简,只有一个圆环 + 大时间 + 一个暂停按钮**。其他信息靠 hover 才显示。

### 3.1 运行中(主态)

```
┌──────────────────────────────────┐
│  [─]                            [×]│  ← 顶部:可拖动条 + 关闭
│                                  │
│           24 : 35               │  ← 64pt Mono 粗体
│                                  │
│      ┌──────────────────┐       │
│      │     ⏱ 进度环      │       │  ← r=120,3px 描边
│      │   (橘→红渐变)    │       │
│      │                  │       │
│      │      ⏸          │       │  ← 居中圆形按钮
│      │                  │       │
│      └──────────────────┘       │
│                                  │
│           重构登录页             │  ← 任务名
│         🍅 第 3 次              │  ← 累计番茄
│                                  │
│   ⏪ 提前结束   ⏱ 调整   ─  ─    │  ← 底部弱化
└──────────────────────────────────┘
```

**视觉:**
- **大圆环**:背景灰细环 + 前景橘→红渐变粗环,顺时针绘制
- **大时间**:SF Mono 64pt,粗 600
- **居中按钮**:`⏸` 大圆按钮(48pt 圆形,蓝底白图)
- **任务名**:SF Pro 14pt 中等
- **统计**:🍅 emoji + 数字
- **底部**:3 个文字按钮,小、灰,弱化

**窗口规格:** 320 × 480,圆角 20,**无标题栏**(自定义 NSPanel + .fullSizeContentView)

**颜色:**
- 圆环渐变:`#FF6B6B`(红) → `#FFA94D`(橘) → `#FFD43B`(黄) 表示专注度
- 按钮:`#4A90E2`(蓝)
- 文字:`#1D1D1F`(主) / `#86868B`(辅)

### 3.2 暂停态

```
┌──────────────────────────────────┐
│  [─]                            [×]│
│                                  │
│           24 : 35               │  ← 时间仍然显示(冻结)
│                                  │
│      ┌──────────────────┐       │
│      │   (圆环停止)     │       │  ← 圆环不再动,但仍显示当前进度
│      │                  │       │
│      │      ▶          │       │  ← 按钮变 ▶
│      │                  │       │
│      └──────────────────┘       │
│                                  │
│            已暂停                │  ← 任务名 位置变「已暂停」灰色
│        ⏱ 暂停于 14:23           │  ← 暂停时间
│                                  │
│   ⏪ 结束并保存   ▶ 继续   ✕ 放弃 │  ← 文字按钮更新
└──────────────────────────────────┘
```

- 圆环保留当前进度,**不再动**
- 居中按钮 ▶ 变 继续
- 任务名位置显示「已暂停」+ 暂停时间点

### 3.3 hover 任务名(显示完整信息)

默认极简,**hover 任务名区域** → 浮出小气泡显示完整信息:

```
        ┌──────────────────────┐
        │ 重构登录页            │
        │ 🔥 紧急 · 进度 75%   │
        │ 今日 2/4 · 累计 1h   │
        └──────────────────────┘
                ↓
        重构登录页
      🍅 第 3 次
```

## 4. 完成态(短暂)

倒计时归零,弹完成态 3 秒后自动开始 5 分钟休息(或点选其他):

```
┌──────────────────────────────────┐
│                                  │
│              ✓                   │  ← 80pt 大对勾
│                                  │
│           专注完成!              │  ← 24pt 粗体
│                                  │
│            🍅 第 3 次            │
│          累计 1h 15m             │
│                                  │
│   ┌────────────┬────────────┐    │
│   │  休息 5 分钟  │  再来一组  │    │
│   └────────────┴────────────┘    │
│                                  │
│         [稍后提醒]               │  ← 文字按钮
│                                  │
└──────────────────────────────────┘
```

**行为:**
- 圆环 0% → 100% 完成,闪绿光一次
- 「✓」图标 fade in + scale 动画
- 「休息 5 分钟」点击 → 转休息倒计时窗口(类似主窗口,绿色调)
- 「再来一组」直接回到启动弹窗,默认上次时长
- 「稍后提醒」关闭,不开始休息

**窗口规格:** 360 × 380

## 8. 快捷键

| 键 | 行为 |
|---|---|
| `Space` | 暂停 / 继续 |
| `Esc` | 关闭窗口(弹确认) |
| `⌘.` | 提前结束 |
| `⌘R` | 重置当前倒计时 |
| `1/2/3` | 启动弹窗内:切预设 15/25/45 |
| `+`/`-` | 启动弹窗:调时间 ±5 分钟 |
| `↩` | 启动弹窗:开始专注 |

## 9. 数据模型(后端)

新表 `pomodoro_sessions`:

| 字段 | 类型 | 备注 |
|---|---|---|
| id | INTEGER PK | |
| task_id | BIGINT | 关联任务(可空) |
| planned_minutes | INT | 计划时长 |
| actual_minutes | INT | 实际时长 |
| status | TINYINT | 0=completed 1=aborted 2=running |
| started_at | BIGINT | 开始 |
| ended_at | BIGINT | 结束(NULL=进行中) |
| note | TEXT | 备注(可选) |

索引:`(task_id, ended_at DESC)`,`(ended_at DESC)`

**API:**
- `POST /api/v1/pomodoros` body `{task_id?, planned_minutes}` → session
- `GET /api/v1/pomodoros?task_id=X&days=7` 历史
- `POST /api/v1/pomodoros/:id/complete` body `{actual_minutes, status}` 标记完成
- `GET /api/v1/pomodoros/stats?days=7` 统计(总次数、总时长)
